>[[일괄처리 방식]]
>- 데이터를 일정량 모으다가 한꺼번에 처리하는 형태
>- 예) OMR 답안지 채점
>- 성능 척도: 시간 당 작업 수행량
>- 개별 요청이 즉각 처리되지 않으므로, 대기시간이 김
>- 입출력에 사람이 개입하지 않음
>- [[보조기억장치]] 등에 [[데이터]]를 모아서 처리 후, [[보조기억장치]]에 저장하거나 [[출력 장치]]로 출력
>
>---
>[[대화식 처리 방식]]
>- 사용자가 명령을 내리면, 즉각적으로 처리 후 결과를 응답
>- 사용자가 편리하게 [[데이터]]를 입력하는 도구가 필요 ([[입력장치]])
>- [[입력장치]]의 발달과 함께 대화형 컴퓨터 방식이 널리 퍼짐
>- 예: 웹, 문서 편집, ATM, POS 등
>- 성능 척도: 빠른 응답 시간
